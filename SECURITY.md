# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest (main) | ✅ |
| Older branches | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do NOT 
open a public issue.

Instead, report it privately by opening a GitHub Security Advisory:
**Settings → Security → Advisories → New Draft Advisory**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix if known

We will respond within 72 hours and work with you on a fix before 
any public disclosure.

## Responsible Use

This tool is intended strictly for authorized penetration testing and 
security research on systems you own or have explicit written permission 
to test.

Unauthorized use against systems you do not own is illegal and unethical. 
The developers of this project accept no liability for misuse.
